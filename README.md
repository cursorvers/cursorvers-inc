# 画像 alt テキスト運用ルール

プロダクト全体でスクリーンリーダー利用者や検索エンジンに十分な文脈を届けるため、下記のルールを必ず守る。

1. **誰が / 何を / どんな目的で** を 1 文で説明する  
   - 例：`alt="Hospital audit team reviewing AI導入データ"`  
   - 単語のみ（例：`alt="Audit"`）は禁止。

2. **ブランド資産は役割まで記述する**  
   - ロゴ：`alt="Cursorvers primary logomark"` のように、どのバージョンかを明示する。

3. **言語はコンテンツに合わせて自然に**  
   - 英語 or 日本語に拘らず、ユーザーにとって最も理解しやすい表現で記述する。

4. **UI変更を伴わないことを前提に即時反映する**  
   - alt テキストの追加・更新はビジュアルへ影響しないため、他タスクを待たずに随時コミットして良い。

5. **新規画像追加時のチェックリスト化**  
   - PR / コミット前に「全画像に上記ルールの alt があるか」を確認する。

このルールを守ることで、アクセシビリティとブランド体験を両立させる。
